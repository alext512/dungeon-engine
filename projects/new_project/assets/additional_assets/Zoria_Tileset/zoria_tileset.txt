Content Notes:
- The pack is a 16x16 Zelda-style tileset and sprite set using an NES-style palette.
- The bundled readme says the overworld content is the most complete, while underworld tiles are unfinished and many sprites are unanimated.
- The bundled readme also says the files in the aseprite folder are layered animations intended to be opened in Aseprite.
